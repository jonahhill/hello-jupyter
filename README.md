# hello-jupyter

A simple introduction to Jupyter notebooks, to demonstrate how cloning works.

## Clone in QuantRocket

CLI:

```shell
quantrocket codeload clone 'hello-jupyter'
```

Python:

```python
from quantrocket.codeload import clone
clone("hello-jupyter")
```
